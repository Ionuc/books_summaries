
Django
======

    - comes with:
        - ORM:
            - Object Relationar Mapping
        - URL Mapping
        - Templates
        - Forms
        - Admin
        - Packages
    - is prefered to install django in a Virtual Environment:
        - python3 -m venv <name>
        - . <name>/bin/activate
        - pip install django
    - you can start the project using:
        - python manage.py
