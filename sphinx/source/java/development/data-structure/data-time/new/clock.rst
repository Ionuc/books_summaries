Clock
=====
- it was added in Java 8
- provides access to the current instance, date and time
- instances of this class are used to find the current instance
- it can be used instead of System.currenttimemillis() method
- the primary purpose of this abstraction is to allow alternate clock to be plugg in as and when requires
- this class helps to test
