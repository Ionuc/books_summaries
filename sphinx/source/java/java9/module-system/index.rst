Module System
=============
    - one of the big java 8 feature is the Module System, a part of Jigsaw Project
    - it was introduced:
        - Modular JDK
        - Modular Java Source Code
        - Modular Run-time Images
        - Encapsulate Java Internal APIs
        - Java Platform Module System
    - before Java 9 versions, applications were using Monolithic Jars to develop Java-Based applications
    - we can use predefined modules or create our own module
